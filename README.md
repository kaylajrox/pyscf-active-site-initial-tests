# pyscf-active-site-initial-tests

The logs and plots folder is how I ended up saving my iterations. The logs are just text files with all the array data used to generate the plots and the metadata of the DFT stuff used - spin, charge, basis, functional, etc. The plots have the same name as the log file (they are timestamped) .
